Black Lister


Black Lister is a browser extension which allows you to highlight pages you like and black out webpages you'd like to never see again. It is especially effective when paired with a search engine.

How to Use

If you don't like a web page, click the big red button. Links to it from any other site will appear fuzzy and grey for as long as it is blammed. Also, the page itself will have a giant mushroom cloud explosion obfuscating it. Should you want to use the site anyways, just click the button in the center of the screen that says "view anyways."

If you really like a web page and want to see links to it more easily, highlight it. Links to it from any other site will be highlighted.

Checking and Editing your Lists

There is a settings section for advanced users. In the settings section you can look at either your "Black List" of blammed sites or your "A List" of highlighted sites. To remove any site from it's list, click the minus button next to it. To add a site to a list, use the input bar at the top.

Advanced Users

When this app checks links to see if they are to be blammed, highlighted, or left alone it checks if the whole name on the list exists in the link. For example www.w3schools.org exists in www.w3schools.org/junk. But if you only wanted the junk page and all subpages to be on a list, you could use a custom list item and narrow down the matching links. For example www.w3schools.org/junk does not exist in www.w3schools.org though it does exist in www.w3schools.org/junk/more_junk.
